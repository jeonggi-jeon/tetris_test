# 네온 테트리스

브라우저에서 실행되는 HTML/CSS/JavaScript 테트리스입니다. 폰에서는 가로폭이 좁을 때 화면 오른쪽에 터치 조작 패널이 표시됩니다.

## 실행 방법

ES 모듈을 사용하므로 **`http://` 또는 `https://`로 페이지를 열어야** 합니다 (`file://` 접속 시 동작하지 않을 수 있습니다).

프로젝트 폴더에서 정적 서버를 띄운 뒤 브라우저에서 예: `http://localhost:8080` 으로 접속합니다.

```bash
npx --yes serve . --listen 8080
```

## 같은 Wi‑Fi에서 휴대폰으로 접속

PC에서 위처럼 서버를 실행한 다음, PC의 사설 IP(예: `192.168.0.12`)를 확인하고 휴대폰 브라우저에서 `http://192.168.0.12:8080` 형태로 접속합니다. Windows 방화벽에서 해당 포트 허용이 필요할 수 있습니다.
